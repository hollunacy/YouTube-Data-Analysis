# YouTube-Data-Analysis
An end-to-end data analytics project focused on extracting, preprocessing, and analyzing YouTube video metrics to classify content, segment channels, and estimate view counts.

## Key Features & Methodology
- **Data Engineering:** Handled missing values and applied Log Transformation to fix highly skewed target variables and mitigate outlier impacts.
- **Exploratory Data Analysis (EDA):** Performed statistical analysis on engagement metrics and audience activity time patterns.
- **Classification & Segmentation:** Developed a video topic classifier testing 5 algorithms (Naive Bayes, Logistic Regression, SVM, KNN, Decision Trees).
- **Segmentation:** Segmented YouTube channels using K-means, DBSCAN, and OPTICS clustering.
- **Advanced Modeling:** Performed regression analysis and time-series exploration to estimate video views. Optimized performance using ensemble methods (Bagging, Boosting, Stacking).

## Tech Stack
- **Language:** Python
- **Libraries:** Pandas, NumPy, Scikit-learn, SciPy
